## EndPoints:
`GET /message`

- Retorna a String salva

`POST /message`

- Salva a string no MongoDB

`GET /live`
- Retorna a String em tempo real

`memory_usage`
- Retorna a quantidade de memória RAM utilizada pelo processo do Node.js
